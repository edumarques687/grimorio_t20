Olá,

Este é um projeto pessoal para criar um Grimório para Tormenta 20. Através dele, você poderá consultar e filtrar a lista de todas as magias já lançadas no sistema, assim como criar listas próprias (necessário logar).

Devo salientar que este projeto AINDA ESTÁ EM DESENVOLVIMENTO, então pode esperar uma repaginada no visual de vez em quando.

Se você tem uma sugestão, encontrou algum erro ou magia faltando, por favor, me envie por e-mail para ermm.rs@gmail.com que vou adicionar/corrigir assim que possível.

P.S.: Este grimorio é atualizado sempre que magias novas são lançadas, mas devido as grandes alterações lançadas na Edição Jogo do Ano, eu criei um clone deste site para manter as magias na versão antiga. Você pode acessar este 'grimorio antigo' através do link na barra de navegação ou clicando aqui.
P.S.2: Todos os dados deste site foram copiados para o outro, então você pode usar o mesmo usuário no outro site e todos os seu grimórios antigos estarão lá, mas grimorios novos e/ou alterações em grimórios existentes não refletirão no outro site nem vice-versa.
